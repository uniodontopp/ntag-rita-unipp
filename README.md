# 📇 Cartão de Visitas Digital

Este é o repositório do meu cartão de visitas pessoal, desenvolvido com foco em performance e estética minimalista.

## 🚀 Link do Projeto
Você pode acessar o cartão clicando aqui: https://uniodontopp.github.io/ntag-rita-unipp

## 🛠️ Tecnologias Utilizadas
* **HTML5/CSS3**: Estrutura e estilização.
* **GitHub Pages**: Hospedagem vitalícia gratuita.
* **NFC Technology**: O link está configurado em uma tag NTAG213 nos crachás de todos colaboradores.

## 🎨 Conceito de Design
O layout segue uma estética de "Bento Grid", inspirada em interfaces modernas de tecnologia, garantindo que as informações de contato sejam acessíveis de forma rápida e visualmente agradável no celular.
